# 🚀 Overview

Zomato Restaurant Management is a **food discovery and restaurant platform** where users can explore food reels uploaded by restaurants and place orders.

The platform connects **food partners and customers** through an engaging food discovery experience.

---

# ✨ Features

• Explore food reels uploaded by restaurants
• Like and save favorite food items
• Restaurant partners can upload food reels
• User authentication system
• Order placement functionality

# 🧪 Installation

Clone repository

```
git clone https://github.com/DHYEY2703/zomato-restaurant-management.git
```

Install dependencies

```
npm install
```

Run backend

```
npm run dev
```

Run frontend

```
npm start
```
